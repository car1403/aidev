# 03. Mini Project Rubric

Supabase 미니 프로젝트, Agent 미니 프로젝트, Workflow 미니 프로젝트에 공통으로 사용할 수 있는 평가 기준입니다.

| 항목 | 배점 | 확인 내용 |
| --- | ---: | --- |
| 문제 정의 | 10 | 사용자가 누구이고 어떤 문제를 해결하는지 명확한가 |
| 구조 설계 | 15 | 전체 흐름, 데이터 흐름, API 또는 Workflow 구조가 설명되는가 |
| 구현 완성도 | 20 | 핵심 기능이 실행되는가 |
| 데이터 저장 | 10 | Supabase 또는 DB 저장 구조가 적절한가 |
| AI 활용 | 15 | AI 응답, Agent, RAG, Workflow 적용 이유가 명확한가 |
| 오류 처리 | 10 | 실패 상황과 fallback이 있는가 |
| 보안/env 관리 | 10 | `.env`, API Key, service role key 관리가 안전한가 |
| 문서/발표 | 10 | README, 테스트, 발표 자료가 정리되어 있는가 |

## 추가 가산 기준

SSE 스트리밍 응답 구현

운영 로그 설계

보안 필터 또는 Guardrail
