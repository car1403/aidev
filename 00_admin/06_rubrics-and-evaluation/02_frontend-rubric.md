# 02. Frontend Rubric

프론트엔드 과제와 프로젝트 평가 기준입니다.

| 항목 | 확인 내용 |
| --- | --- |
| Streamlit 화면 | 사용자가 입력하고 결과를 볼 수 있는가 |
| API 호출 | FastAPI API를 안정적으로 호출하는가 |
| 상태 관리 | session_state를 적절히 사용하는가 |
| 오류 표시 | API 실패 시 사용자에게 안내하는가 |
| 보안 | service role key를 화면 코드에 넣지 않았는가 |
| 사용성 | 버튼, 입력창, 결과 영역이 이해하기 쉬운가 |
