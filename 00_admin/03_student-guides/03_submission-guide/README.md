# 03_submission-guide

과제와 프로젝트 제출 안내입니다.

## 제출 전 확인

```text
[ ] README.md가 있다.
[ ] 실행 방법이 적혀 있다.
[ ] requirements.txt가 있다.
[ ] .env.example이 있다.
[ ] 실제 .env는 제출하지 않는다.
[ ] .venv는 제출하지 않는다.
[ ] 테스트 체크리스트가 있다.
[ ] 발표 자료가 있다.
```

## 제출하면 안 되는 것

```text
.env
.venv
__pycache__
개인 API Key
실제 개인정보
회사 내부 문서 원본
```

## README에 들어갈 내용

```text
프로젝트 이름
프로젝트 목적
사용 기술
실행 방법
환경변수 안내
주요 기능
테스트 방법
팀원 역할
```
