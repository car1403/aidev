# 03_student-guides

수업 참여자가 직접 보고 따라 하는 안내서입니다.

이 폴더는 수업 첫날부터 사용합니다. Python, VS Code, PowerShell, `.venv`, Markdown Preview 같은 기본 환경을 준비한 뒤 01~09 과정을 진행합니다.

## 읽는 순서

```text
01_getting-started
-> 02_how-to-study
-> 03_submission-guide
```

## 가장 먼저 할 일

```text
1. Python 설치 확인
2. VS Code 설치 확인
3. VS Code Extensions 설치
4. C:\aidev 폴더 열기
5. README.md를 Markdown Preview로 보기
6. PowerShell에서 python --version 확인
7. 첫 과정 폴더에서 .venv 만들기
```

## 핵심 요약

```text
README.md는 설명서입니다.
requirements.txt는 필요한 패키지 목록입니다.
.env.example은 환경변수 예시입니다.
.env는 실제 키를 넣는 개인 파일입니다.
.venv는 프로젝트별 Python 실행 환경입니다.
```
