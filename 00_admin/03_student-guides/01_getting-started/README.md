# Getting Started

처음 환경을 준비할 때 보는 문서입니다.

## 준비 순서

```text
1. GitHub 계정 만들기
2. Python 설치 확인
3. VS Code 설치
4. VS Code에서 GitHub 로그인
5. GitHub Copilot Chat 준비
6. VS Code 확장 설치
7. PowerShell 기본 명령 확인
8. Markdown Preview 사용법 확인
9. .venv 만들기와 requirements.txt 설치
10. 첫 예제 실행
```

## 첫 실행 전 확인

- [ ] GitHub 계정을 만들었다.
- [ ] Python이 설치되어 있다.
- [ ] VS Code가 설치되어 있다.
- [ ] VS Code에서 `C:\aidev` 폴더를 열 수 있다.
- [ ] VS Code에서 GitHub 계정으로 로그인했다.
- [ ] GitHub Copilot Chat 사용 준비를 확인했다.
- [ ] README.md Preview를 열 수 있다.
- [ ] PowerShell에서 `python --version`을 실행할 수 있다.
