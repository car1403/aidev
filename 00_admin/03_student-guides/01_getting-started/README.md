## 오늘 준비가 끝났다는 기준

아래 항목이 모두 가능하면 준비가 끝난 것입니다.

```text
[ ] PowerShell에서 python --version을 실행할 수 있다.
[ ] PowerShell에서 pip --version을 실행할 수 있다.
[ ] VS Code에서 C:\aidev 폴더를 열 수 있다.
[ ] VS Code에서 README.md Preview를 열 수 있다.
[ ] 과정 폴더에서 .venv를 만들 수 있다.
[ ] .venv를 활성화할 수 있다.
[ ] pip install -r requirements.txt를 실행할 수 있다.
[ ] Codex 설치 방식과 로그인 방식을 이해했다.
[ ] ChatGPT 구독 결제와 API 사용 결제의 차이를 이해했다.
```

## Codex와 결제 안내를 따로 보는 이유

Codex와 OpenAI 결제는 수업 환경, 개인 계정, 학교/회사 계정, ChatGPT 구독, API 사용 여부에 따라 달라질 수 있습니다.

그래서 수업에서는 다음 원칙을 지킵니다.

```text
1. 설치와 결제는 강사가 안내한 방식만 따른다.
2. 개인 결제 정보는 수업 중 공유하지 않는다.
3. API Key와 결제 정보는 코드나 문서에 적지 않는다.
4. 유료 기능은 반드시 본인 또는 기관의 승인 후 사용한다.
```

## 수업 중 막히면 먼저 확인할 것

```text
1. 현재 PowerShell 위치가 맞는가?
2. .venv가 활성화되어 있는가?
3. requirements.txt가 있는 폴더에서 명령을 실행하고 있는가?
4. .env.example을 .env로 복사했는가?
5. 오류 메시지를 끝까지 읽었는가?
```
