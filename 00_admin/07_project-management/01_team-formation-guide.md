# 01. Team Formation Guide

팀 구성 안내입니다.

## 팀 구성

```text
4~5명
```

## 역할 예시

```text
Project Lead:
주제, 일정, 발표 흐름 관리

Backend:
FastAPI, Supabase, Agent API 담당

Frontend:
Streamlit, UI, API 호출 담당

Docs/QA:
README, 테스트 체크리스트, 발표 자료 담당
```

작은 팀에서는 한 사람이 여러 역할을 맡아도 됩니다.

## 팀 구성 시 주의

```text
숙련된 참여자와 입문 참여자가 함께 설명하며 진행하도록 구성합니다.
역할을 나누되 모든 팀원이 전체 흐름을 설명할 수 있어야 합니다.
```
