## 핵심 요약

09 과정은 새 도구를 더 많이 배우는 시간이 아니라, 08에서 배운 내용을 하나의 결과물로 묶는 시간입니다.

팀 프로젝트에서 중요한 질문은 다음과 같습니다.

```text
무엇을 자동화할 것인가?
왜 자동화가 필요한가?
어떤 입력이 들어오는가?
어떤 조건으로 분기되는가?
AI는 무엇을 판단하거나 생성하는가?
AIPP, n8n, Dify 중 어떤 도구가 가장 적합한가?
실패했을 때 어떻게 처리할 것인가?
최종 발표에서 무엇을 보여줄 것인가?
```

강의 중에는 `05_project-class-manual.md`를 열어 두고 팀별 진행 상황을 점검하면 됩니다.
