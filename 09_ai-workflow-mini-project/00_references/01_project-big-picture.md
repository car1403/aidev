# 01. Project Big Picture

## 프로젝트 큰 그림

09 과정에서는 기술 지원 문의를 자동으로 처리하는 AI 워크플로우를 만듭니다.

```text
문의 접수
-> 유형과 긴급도 분류
-> 관련 문서 검색
-> 답변 초안 생성
-> 품질 검증
-> 운영팀 알림 또는 답변 후보 생성
-> 이벤트와 지표 기록
```

## 사용할 수 있는 도구

```text
AIPP: 워크플로우 설계
n8n: Webhook/API 자동화 연결
Dify: AI 앱과 Knowledge/RAG
Python FastAPI: API 샘플 구현
Streamlit: 시연 화면
```
