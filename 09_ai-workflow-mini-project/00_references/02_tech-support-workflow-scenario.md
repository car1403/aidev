# 02. Tech Support Workflow Scenario

## 기본 시나리오

고객이 기술 지원 문의를 남기면 AI 워크플로우가 자동으로 문의를 분석합니다.

## 필수 기능

```text
1. 문의 입력
2. 유형 분류
3. 긴급도 판단
4. 관련 문서 매칭
5. 답변 초안 생성
6. 답변 품질 검증
7. 다음 Action 결정
8. 실행 이벤트 기록
```

## 확장 기능

```text
n8n Webhook으로 문의 접수
Dify Knowledge로 문서 검색
AIPP에서 노드 기반 설계
운영 로그 기반 개선 제안
```
