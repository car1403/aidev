# 04. Team Project Checklist

팀 프로젝트 진행 체크리스트입니다.

## 시작 전

```text
[ ] 팀 주제 확정
[ ] 사용자 정의
[ ] 사용할 도구 선택
[ ] 팀원 역할 분담
[ ] 일정 확정
[ ] 샘플 프로젝트 실행 확인
```

## 설계 중

```text
[ ] Trigger 정의
[ ] Input 데이터 정의
[ ] Condition 정의
[ ] Action 정의
[ ] AI 역할 정의
[ ] Tool/API/Knowledge/RAG 역할 정의
[ ] 데이터 처리 노드 정의
[ ] 노드 간 데이터 흐름 정의
[ ] 오류 처리 설계
[ ] 로그/지표 설계
```

## 운영/보안 검토

```text
[ ] Prompt Injection 방어 기준 작성
[ ] 입력 검증 기준 작성
[ ] 출력 필터링 기준 작성
[ ] API Key 노출 방지 확인
[ ] 비용/API 사용량 관리 기준 작성
[ ] 리소스 관리와 확장성 고려사항 작성
[ ] Multi-Agent Workflow 적용 여부 작성
```

## 발표 전

```text
[ ] 데모 시나리오 준비
[ ] 정상 입력 테스트
[ ] 긴급 입력 테스트
[ ] 오류 입력 테스트
[ ] 보안 입력 테스트
[ ] 발표 자료 작성
[ ] 시연 순서 확인
```
