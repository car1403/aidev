- 기본 실습은 mock 응답으로 진행합니다.

- 백엔드 API 연결은 `03_api-integration/00_sample_backend`의 `/api/chat/mock`을 기준으로 먼저 연습합니다.
- 로그인 상태, 사용자별 대화 이력, 서비스 로그는 `05_state-session-and-data`에서 본격적으로 다룹니다.
