# Lab 04. Final UI Review

## 목표

최종 프론트엔드 프로젝트를 제출하기 전에 UI 흐름과 보안 기준을 점검합니다.

## 체크리스트

- [ ] 앱이 `streamlit run`으로 실행된다.
- [ ] `API_BASE_URL`을 사용해 백엔드를 호출한다.
- [ ] 백엔드 연결 실패 메시지가 있다.
- [ ] 로딩 상태가 보인다.
- [ ] 챗봇 UI가 있다.
- [ ] 대화 이력 또는 서비스 로그 조회 화면이 있다.
- [ ] 프론트엔드에 service role key나 LLM API key가 없다.
- [ ] 실행 결과와 오류 해결 기록이 문서에 있다.
