# Review Report Example

## 리뷰 대상

`01_review-target-simple-app.py`

## 발견한 점

- 이름이 비어 있을 때 안내 메시지가 표시된다.
- 점수 기준은 80점으로 명확하다.
- 입력 영역과 결과 영역이 같은 화면에 있어 이해하기 쉽다.

## 개선 아이디어

- 점수 기준을 사이드바에서 조정할 수 있게 만들 수 있다.
- 결과 메시지를 함수로 분리할 수 있다.
- 제출 전 입력값 확인을 별도 함수로 만들 수 있다.

## 최종 판단

현재 코드는 입문 예제로 적절하지만, 반복 사용을 고려하면 함수 분리가 필요하다.
