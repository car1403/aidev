# -----------------------------------------------------------------------------
# Streamlit 화면 구성 패키지
# -----------------------------------------------------------------------------

# Python 코드만으로 웹 화면을 빠르게 만들기 위한 UI 프레임워크입니다.

# - frontend/pages/01_chatbot.py: 챗봇 입력/응답 화면
# - frontend/pages/02_chat_history.py: 대화 이력 조회 화면
# - frontend/pages/03_service_logs.py: 서비스 로그 조회 화면
# - frontend/pages/04_deployment_check.py: 배포 전 점검 화면

# -----------------------------------------------------------------------------
# 화면용 데이터 표시 패키지

# 사용 위치:
# - frontend/pages/02_chat_history.py: 대화 이력을 표로 표시
# - frontend/pages/03_service_logs.py: 서비스 로그를 표와 차트로 표시
pandas==2.2.3


# -----------------------------------------------------------------------------
# FastAPI 백엔드 호출 패키지
# -----------------------------------------------------------------------------

# Streamlit 프론트엔드에서 FastAPI 백엔드 API를 호출하기 위한 HTTP 클라이언트입니다.

# - frontend/app.py: /health API를 호출해 백엔드 상태 확인

# 외부의 C:\aidev\.env 또는 C:\aidev\03_supabase-ai-frontend\.env를 읽는 구조가 아닙니다.

# - frontend/app.py: API_BASE_URL 읽기
# - frontend/pages/*.py: 각 화면에서 API_BASE_URL 읽기
