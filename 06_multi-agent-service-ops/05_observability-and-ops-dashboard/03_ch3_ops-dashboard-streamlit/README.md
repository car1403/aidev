# 03_ch3_ops-dashboard-streamlit

Streamlit으로 간단한 운영 대시보드를 만드는 방법을 학습합니다.

## 대시보드에 표시할 정보

- 서비스별 상태
- 최근 이벤트 로그
- Agent 실행 이력
- 실패 건수
- 복구 조치 상태

## 실행

```powershell
streamlit run.\03_ch3_ops-dashboard-streamlit\01_ops-dashboard.py --server.port 8803
```

브라우저에서 확인합니다.

```text
http://127.0.0.1:8803
```
