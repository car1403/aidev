# 20 Assignments

## 과제. Docker Compose 기반 AI 서비스 실행 보고서

Docker Compose로 backend, frontend, worker, monitor를 실행하고 운영 관점에서 결과를 정리합니다.

## 제출 내용

- 서비스 구성도
- `docker compose config` 결과 요약
- `docker compose ps` 결과
- backend `/health` 확인 결과
- backend/worker/monitor 로그 확인 결과
- GitHub Actions 자동 검증 계획
- AWS 배포 전 체크리스트

## 평가 기준

- Docker Compose 구조를 설명할 수 있는가?
- 서비스별 역할이 명확한가?
- health check와 logs를 확인했는가?
- GitHub Actions와 AWS 배포 흐름을 운영 관점으로 연결했는가?
