# 20_assignments

`01_python-basic` 과제 모음입니다.

Lab은 수업 중 함께 따라 하며 문법을 익히는 실습이고, Assignment는 같은 문법을 스스로 조합해 보는 과제입니다.

## 제출 기준

과제 파일은 `20_assignments\submissions` 폴더에 작성합니다.

```powershell
mkdir .\01_python-basic\20_assignments\submissions
```

파일명 예시:

```text
assignment01_profile.py
assignment02_grade.py
assignment03_menu_calculator.py
```

## 과제 목록

| 순서 | 파일 | 핵심 내용 |

| 1 | `assignment01_python-start-and-variables.md` | 출력, 입력, 변수, 자료형 |

| 7 | `assignment07_basic-python-final.md` | 기초 문법 종합 미니 프로그램 |

## 공통 완료 기준

모든 과제는 아래 기준을 만족해야 합니다.

```text
1. 실행 가능한 Python 파일을 제출합니다.
2. 코드 상단에 과제 제목과 작성 목적을 주석으로 적습니다.
3. 주요 코드에는 초보자가 이해할 수 있는 주석을 작성합니다.
4. 실행 결과를 README 또는 주석으로 간단히 기록합니다.
5. 오류가 발생한 경우 원인과 해결 방법을 함께 정리합니다.
```
