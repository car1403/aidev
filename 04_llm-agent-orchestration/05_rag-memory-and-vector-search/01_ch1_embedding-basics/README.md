# 01_ch1_embedding-basics

이 챕터에서는 문장을 숫자 벡터로 바꾸는 Embedding 개념을 학습합니다.

## 핵심 개념

- Embedding은 문장의 의미를 숫자 목록으로 표현합니다.
- 의미가 비슷한 문장은 벡터 공간에서도 가깝게 위치합니다.
- 벡터 유사도는 RAG 검색의 출발점입니다.
