# Lab 01. First OpenAI Call

## 목표

`gpt-4.1-mini` 모델을 사용해 OpenAI API 첫 응답을 받아봅니다.

## 진행

1. `.env.example`을 `.env`로 복사합니다.
2. `OPENAI_API_KEY`를 입력합니다.
3. `OPENAI_MODEL=gpt-4.1-mini`인지 확인합니다.
4. `01_check-env.py`를 실행합니다.
5. `02_openai-basic-response.py`를 실행합니다.

## 제출

- 실행한 명령
- 출력된 응답 요약
- 오류가 있었다면 해결 과정
