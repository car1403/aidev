# Lab 03. Compare Two Models

## 목표

OpenAI `gpt-4.1-mini`와 로컬 Llama의 응답을 비교합니다.

## 진행

1. OpenAI API Key를 준비합니다.
2. Ollama/Llama 컨테이너를 실행합니다.
3. `01_compare-basic-answer.py`를 실행합니다.
4. `02_compare-prompt-style.py`를 실행합니다.
5. 결과를 표로 정리합니다.

## 비교 기준

- 응답 속도
- 문체
- 정확도
- 비용 또는 실행 자원
- 수업용 예제로 적합한 정도
