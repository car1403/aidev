# 02_how-to-study

이 폴더는 수업을 어떻게 따라가면 좋은지 안내합니다.

## 추천 학습 방법

```text
1. README.md를 Preview로 먼저 읽습니다.
2. 폴더 구조를 확인합니다.
3. 실행 명령어를 한 줄씩 따라 합니다.
4. 정상 결과를 캡처하거나 메모합니다.
5. 오류가 나면 오류 메시지를 복사해 원인을 확인합니다.
6. 실습 후 자기 말로 오늘 배운 내용을 3줄로 정리합니다.
```

## AI 도구를 사용할 때

AI에게 바로 "다 만들어줘"라고 하기보다 아래처럼 질문합니다.

```text
이 코드가 어떤 흐름인지 초보자에게 설명해줘.
이 오류 메시지의 원인을 단계별로 알려줘.
이 함수에 주석을 초보자용으로 추가해줘.
이 API를 테스트하는 방법을 알려줘.
```

## 복습 방법

```text
1. 오늘 실행한 명령어 다시 실행
2. README의 체크리스트 다시 확인
3. 코드 주석 읽기
4. 오류를 일부러 만들어 보고 고치기
5. 배운 내용을 작은 예제로 다시 작성
```
