## 주제 선택 기준

- 과정에서 배운 기술을 실제로 연결할 수 있는가?
- README만 보고 실행할 수 있는가?
- 입력, 처리, 출력 흐름이 분명한가?
- 로그와 오류 처리 기준이 있는가?
- `.env`와 API Key 보안 기준을 지키는가?
- 발표 시 시연 가능한 결과가 있는가?
