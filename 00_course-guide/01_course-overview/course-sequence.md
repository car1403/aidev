# Course Sequence

이 과정은 Python 기초에서 시작해 AI 서비스 개발, LLM Agent, 멀티 에이전트 서비스 운영 프로젝트까지 이어집니다.

| 순서 | 과정 | 핵심 역할 |
| --- | --- | --- |
| 01 | Python/Git Foundation | Python 문법, 프로젝트 구조, Git/GitHub 기초 |
| 02 | Supabase AI Backend | FastAPI, Gemini API, Supabase DB/Auth/RLS, 백엔드 API |
| 03 | Supabase AI Frontend | Streamlit UI, 백엔드 API 호출, 상태 관리 |
| 04 | Supabase AI Mini Project | FastAPI, Streamlit, Supabase를 연결한 팀 프로젝트 |
| 05 | LLM Agent Orchestration | OpenAI/Ollama, Tool Use, RAG, Memory, LangGraph |
| 06 | LLM Agent Mini Project | 단일 Agent 추론, 도구 선택, 오류 감지, 결과 검증 |
| 07 | Multi-Agent Service Ops | Docker Compose, GitHub Actions, AWS 선택 배포, 운영/보안 |
| 08 | Multi-Agent Service Mini Project | 멀티 에이전트 협업과 장애 복구 서비스 프로젝트 |

## 흐름

```text
01 Python/Git
-> 02 Backend
-> 03 Frontend
-> 04 Supabase AI Mini Project
-> 05 LLM Agent
-> 06 Agent Mini Project
-> 07 Service Ops
-> 08 Multi-Agent Service Mini Project
```

`04`, `06`, `08`은 프로젝트 과정입니다. 이 과정에서는 기능 구현뿐 아니라 설계 문서와 결과 보고서를 함께 정리합니다.
