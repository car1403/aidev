# 01 Python/Git Foundation Plan

이 계획서는 `01_python-git-foundation` 과정의 수업 흐름을 정리합니다.

## 목표

- Python 기초 문법을 읽고 실행할 수 있습니다.
- 함수, 모듈, 예외 처리, OOP, 테스트의 기본 구조를 이해합니다.
- Git/GitHub와 VS Code Source Control로 변경 이력을 관리할 수 있습니다.
- 이후 백엔드 과정에서 필요한 최소 개발 습관을 갖춥니다.

## 진행 흐름

| 순서 | 단원 | 핵심 내용 |
| --- | --- | --- |
| 1 | `01_python-basic` | 변수, 자료형, 입출력, 조건문, 반복문, 함수, 파일/JSON |
| 2 | `02_python-advanced` | 함수 심화, 모듈/패키지, 예외 처리, OOP, 테스트, 프로젝트 구조 |
| 3 | `03_git-github` | Git 기본 명령, 커밋/브랜치, VS Code Source Control, README, `.gitignore` |

## 산출물

- Python 기초 미니 프로젝트 실행 결과
- Python 심화 미니 프로젝트 또는 테스트 실행 결과
