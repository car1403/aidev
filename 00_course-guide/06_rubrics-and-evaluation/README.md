## 공통 평가 기준

- README만 보고 실행할 수 있는가?
- `.env.example`, `requirements.txt`, 실행 방법이 정리되어 있는가?
- 코드와 문서가 같은 구조를 설명하는가?
- 핵심 기능이 실제로 동작하는가?
- 보안상 노출되면 안 되는 정보가 없는가?
- 프로젝트 산출물과 발표 내용이 일치하는가?
