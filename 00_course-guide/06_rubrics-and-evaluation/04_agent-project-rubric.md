# 04. Agent Project Rubric

LLM Agent 및 Multi-Agent 프로젝트 평가 기준입니다.

| 항목 | 확인 내용 |
| --- | --- |
| Agent 역할 | Agent가 맡는 일이 명확한가 |
| Tool 사용 | Tool 호출 이유와 입력/출력이 설명되는가 |
| Memory/RAG | Memory 또는 RAG 사용 기준이 있는가 |
| Workflow | 단계별 흐름과 상태 전이가 명확한가 |
| 검증 | Agent 결과를 검토하거나 개선하는 구조가 있는가 |
| 운영 | 로그, tracing, fallback 기준이 있는가 |
