# 01. Backend Rubric

백엔드 과제와 프로젝트 평가 기준입니다.

| 항목 | 확인 내용 |
| --- | --- |
| FastAPI 구조 | app, router 또는 endpoint가 명확한가 |
| Pydantic 검증 | 요청 데이터 검증이 있는가 |
| Supabase 연동 | 테이블 조회/저장 흐름이 동작하는가 |
| 보안 | service role key를 서버에서만 사용하는가 |
| 오류 처리 | HTTPException 또는 fallback이 있는가 |
| 문서화 | 실행 방법과 API 설명이 있는가 |
