# 06. Final Presentation Rubric

최종 발표 평가 기준입니다.

| 항목 | 배점 | 확인 내용 |
| --- | ---: | --- |
| 문제 설명 | 15 | 해결하려는 문제가 이해되는가 |
| 구조 설명 | 20 | 백엔드, 프론트엔드, DB, AI 흐름을 설명하는가 |
| 시연 | 20 | 실제 실행 또는 설계 결과를 보여주는가 |
| 기술 선택 이유 | 15 | 도구 선택 이유가 타당한가 |
| 오류/보안/운영 | 15 | 오류 처리, 보안, 비용, 로그 기준이 있는가 |
| 발표 완성도 | 15 | 시간 내에 명확하게 발표하는가 |

## 발표 중 반드시 들을 질문

```text
왜 이 도구를 선택했나요?
실패하면 어떻게 처리하나요?
데이터는 어디에 저장되나요?
API Key는 어디에 보관하나요?
운영한다면 어떤 로그를 보나요?
```
