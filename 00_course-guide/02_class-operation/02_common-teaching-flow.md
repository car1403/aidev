# 02. Common Teaching Flow

모든 차시에 공통으로 사용할 수 있는 수업 흐름입니다.

## 수업 시작

```text
1. 오늘 배울 내용 한 문장으로 설명
2. 오늘 사용할 폴더 안내
3. README.md Preview 열기
4. 오늘 실행할 명령어 목록 확인
5..venv 활성화 확인
```

## 개념 설명

```text
1. 기술 이름
2. 왜 필요한지
3. 어디에서 쓰이는지
4. 오늘 실습에서는 어떤 파일에서 확인하는지
```

## 실습 진행

```text
1. 먼저 실행
2. 같은 명령어 실행
3. 정상 결과 화면 확인
4. 일부러 작은 오류를 보여주고 해결
5. 체크리스트에 완료 표시
```

## 수업 마무리

```text
1. 오늘 만든 것 확인
2. 오늘 배운 키워드 정리
3. 자주 난 오류 정리
4. 다음 차시와 연결 설명
5. 과제 또는 복습 안내
```

## 학습 질문을 받을 때

실습 중 오류가 발생하거나 실행이 되지 않을 때는 아래 순서로 확인합니다.

```text
1. 어떤 명령어를 실행했나요?
2. 현재 폴더가 어디인가요?
3..venv가 활성화되어 있나요?
4. 오류 메시지 전체를 보여주세요.
5. README의 어떤 단계에서 막혔나요?
```
