현재 과정은 `01`부터 `08`까지 진행합니다.

6. [가상환경(.venv)과 pip](./03_student-guides/01_getting-started/05_venv-and-pip-guide.md)를 확인합니다.

## 핵심 요약

- `01`에서는 Python과 Git/GitHub 기초를 별도 과정으로 먼저 정리합니다.
- `02`부터 `04`까지는 Supabase 기반 웹 서비스와 AI 챗봇 흐름을 학습합니다.
- `05`부터는 Docker 기반 로컬 환경에서 LLM Agent와 RAG, Tool Use를 다룹니다.
- `07`에서는 Docker Compose, AWS, GitHub Actions, 운영 모니터링, 보안 가드레일을 학습합니다.
- `08`에서는 멀티 에이전트 서비스 운영 미니 프로젝트를 진행합니다.
- VS Code의 AI Chat은 GitHub Copilot Chat을 기준으로 안내합니다.

## 문서 사용 방법

필요한 설명은 링크를 따라 바로 이동하시면 됩니다. 예를 들어 VS Code 확장 프로그램이 필요하면 [VS Code 확장 프로그램과 GitHub Copilot Chat](./03_student-guides/01_getting-started/03_vscode-extensions-guide.md)을 열고, 특정 과정의 설치 방법이 필요하면 각 과정의 `SETUP.md`를 확인합니다.

문서를 수정할 때는 루트 [README](../README.md)와 이 문서의 링크가 함께 맞는지 확인하면 전체 과정 탐색이 훨씬 쉬워집니다.
