# 02. During Class Checklist

수업 중 확인할 체크리스트입니다.

## 시작 시

```text
[ ] 오늘 사용할 폴더를 안내했다.
[ ] README.md를 Preview로 열었다.
[ ] 오늘의 목표를 한 문장으로 설명했다.
[ ].venv 활성화 여부를 확인했다.
```

## 실습 중

```text
[ ] 먼저 실행했다.
[ ] 직접 같은 명령어를 실행했다.
[ ] 정상 출력 결과를 확인했다.
[ ] 오류가 난 현재 폴더를 확인했다.
[ ] 오류 메시지를 끝까지 읽었다.
```

## 마무리

```text
[ ] 오늘 만든 파일 또는 실행 결과를 확인했다.
[ ] 오늘 배운 키워드를 정리했다.
[ ] 다음 수업과 연결되는 부분을 설명했다.
[ ] 과제 또는 복습 항목을 안내했다.
```
