# 01. Python Install Check

Python 설치 문제를 확인하는 문서입니다.

## 확인 명령어

```powershell
python --version
py --version
pip --version
```

## 정상 기준

```text
Python 3.12.x 또는 3.11.x
pip 버전 표시
```

## 문제 해결

```text
python 명령이 안 되면 py --version을 확인합니다.
둘 다 안 되면 Python을 다시 설치합니다.
설치 시 Add python.exe to PATH를 체크합니다.
PowerShell을 닫고 다시 엽니다.
```
