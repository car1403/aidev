# Agent Project Idea List

팀 프로젝트 주제 예시입니다.

## 추천 주제

- 일정 조정 에이전트
- 학습 코치 에이전트
- 문서 검색 Q&A 에이전트
- 고객 문의 분류 에이전트
- 개인 업무 계획 에이전트
- 회의록 요약 및 액션 아이템 추출 에이전트
- 개발 오류 분석 도우미 에이전트

## 주제 선택 기준

좋은 주제는 다음 조건을 만족합니다.

- 사용자의 요청 예시가 명확하다.
- 최소 2개 이상의 Tool이 필요하다.
- 결과를 화면에서 확인하기 쉽다.
- 팀원이 역할을 나누어 구현할 수 있다.
- 하루 안에 Mock data로 먼저 완성할 수 있다.
