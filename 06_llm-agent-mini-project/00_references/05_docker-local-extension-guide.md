# 05. Docker Local Extension Guide

`06_llm-agent-mini-project`에서는 단일 Agent 프로젝트 완성에 집중합니다. Docker는 필수 범위가 아니라, 로컬 실행을 조금 더 안정적으로 만들기 위한 선택 확장으로만 다룹니다.

## 이 과정에서의 Docker 기준

```text
필수:
Python .venv
Agent 실행 코드
테스트와 결과 보고서

선택:
단일 컨테이너 실행
Ollama 또는 pgvector 연결
```

여러 서비스를 묶는 Docker Compose, Health Check, Restart Policy, AWS 배포, GitHub Actions는 07 과정에서 다룹니다.

## 기억할 문장

```text
06에서는 Agent의 판단 구조와 검증 흐름이 핵심입니다.
서비스 운영과 배포 자동화는 07에서 확장합니다.
```
