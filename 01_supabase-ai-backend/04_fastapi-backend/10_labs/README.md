# 10_labs

FastAPI 수업 중 진행하는 실습입니다.

## 실습 목록

```text
lab01_hello-fastapi
lab02_path-query-params
lab03_request-body-validation
lab04_response-model
lab05_async-external-api
lab06_in-memory-crud-api
lab99_mini-api-server
```

## 권장 진행 순서

1. `lab01_hello-fastapi`에서 서버 실행과 `/docs` 확인
2. `lab02_path-query-params`에서 path/query parameter 연습
3. `lab03_request-body-validation`에서 request body와 Pydantic 검증 연습
4. `lab04_response-model`에서 응답 모델과 민감정보 제외 확인
5. `lab05_async-external-api`에서 async 흐름 이해
6. `lab06_in-memory-crud-api`에서 CRUD API 전체 흐름 연습
7. `lab99_mini-api-server`에서 작은 API 서버 완성

각 lab은 `starter.py`를 먼저 열어 TODO를 채우고, 막히면 `solution.py`와 비교합니다.
