# 01. Requirements

백엔드 미니 서비스가 해결할 문제와 필수 기능을 정리합니다.

## 수업용 서비스 주제

```text
AI 질문 응답 백엔드 미니 서비스
```

사용자가 질문을 보내면 백엔드가 답변을 만들고, 질문/답변/로그를 저장하는 서비스입니다.

## 필수 기능

| 기능 | 설명 |
| --- | --- |
| 상태 확인 | `/health`로 서버 실행 상태를 확인합니다. |
| 질문 등록 | 사용자의 질문을 받아 AI 답변을 생성합니다. |
| 질문/답변 저장 | 질문과 답변을 하나의 기록으로 저장합니다. |
| 기록 조회 | 저장된 질문/답변 목록을 조회합니다. |
| 서비스 로그 저장 | API 처리 성공/실패를 로그로 남깁니다. |
| 오류 응답 | 잘못된 요청에 대해 명확한 오류 메시지를 반환합니다. |

## 수업 중 결정할 것

- 사용자 id를 어떻게 받을 것인가?
- 질문 길이는 몇 글자까지 허용할 것인가?
- mock LLM 답변과 실제 LLM 답변을 어떻게 구분할 것인가?
- Supabase에 어떤 테이블을 만들 것인가?
- 서비스 로그에 어떤 metadata를 남길 것인가?
