# Lab 99. Final Checklist

## 목표

미니 서비스 실습 결과를 최종 점검합니다.

## 점검 항목

| 항목 | 확인 |
| --- | --- |
| 요구사항 | 서비스 목적, 필수 기능, 제외 범위가 정리되어 있나요? |
| API 설계 | `/questions`와 `/service-logs` API가 명확히 설계되어 있나요? |
| Supabase 스키마 | `mini_questions`, `mini_service_logs`가 생성되어 있나요? |
| mock 서버 | `main_mock.py`에서 질문 생성과 조회가 되나요? |
| Supabase 서버 | `main_supabase.py`에서 실제 저장이 되나요? |
| 서비스 로그 | 성공/실패 로그를 확인할 수 있나요? |
| 오류 응답 | 데이터 없음, 저장 실패 같은 상황을 구분할 수 있나요? |
| 보안 | API Key, 토큰, 비밀번호를 로그에 저장하지 않나요? |

## 제출 또는 정리할 내용

- 실행한 API 목록
- 생성한 Supabase 테이블 이름
- 성공한 요청 예시
- 실패한 요청 예시
- 서비스 로그 예시
- 개선하고 싶은 점
