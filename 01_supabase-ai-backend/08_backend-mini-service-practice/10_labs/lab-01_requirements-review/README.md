# Lab 01. Requirements Review

## 목표

미니 서비스가 해결할 문제와 필수 기능을 확인합니다.

## 확인 파일

```text
../01_requirements/service-requirements.md
```

## 확인할 내용

- 서비스 이름이 명확한가요?
- 서비스 목적을 한 문장으로 설명할 수 있나요?
- 필수 기능과 선택 기능이 구분되어 있나요?
- Supabase에 저장할 데이터와 Upstash Redis로 처리할 데이터가 구분되어 있나요?
- 제외 범위가 명확한가요?

## 정리할 내용

아래 질문에 답을 적어 봅니다.

```text
1. 이 미니 서비스는 어떤 문제를 해결하나요?
2. 가장 중요한 API 기능은 무엇인가요?
3. 이번 단원에서 구현하지 않는 것은 무엇인가요?
```
