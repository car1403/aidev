# 01. LLM API Concepts

LLM API가 무엇인지, 웹 서비스 백엔드에서 왜 모델 API를 호출하는지 학습합니다.

## 핵심 질문

- ChatGPT 웹 화면에서 질문하는 것과 API로 호출하는 것은 무엇이 다른가요?
- 모델, prompt, message, token은 각각 무엇인가요?
- 왜 API key가 필요한가요?
- 왜 실제 API 호출 전 mock 예제로 구조를 먼저 배우나요?

## 실행 예제

```powershell
cd C:\aidev\01_supabase-ai-backend
.\.venv\Scripts\Activate.ps1
python .\05_llm-api-integration\01_llm-api-concepts\01_message_structure.py
```
