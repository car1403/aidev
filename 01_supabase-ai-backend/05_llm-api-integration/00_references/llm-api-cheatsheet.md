# LLM API Cheatsheet

LLM API는 코드에서 AI 모델을 호출하는 방법입니다.

## 기본 흐름

```text
사용자 질문
-> Python 또는 FastAPI 코드
-> LLM API 요청
-> 모델 응답
-> 응답 정리
-> 사용자에게 반환
```

## 주요 용어

| 용어 | 설명 |
| --- | --- |
| Model | 응답을 생성하는 AI 모델입니다. |
| Prompt | 모델에게 보내는 지시문이나 질문입니다. |
| Message | role과 content로 구성된 대화 단위입니다. |
| Token | 모델이 입력과 출력을 처리하는 단위입니다. |
| Temperature | 응답의 다양성과 창의성을 조절합니다. |
| Max tokens | 모델이 생성할 수 있는 최대 출력 길이를 제한합니다. |

## 싱글턴과 멀티턴

```text
싱글턴
-> 현재 질문 하나만 보냄
-> 이전 대화 문맥 없음

멀티턴
-> 이전 대화 이력을 함께 보냄
-> 문맥을 이어서 답변 가능
```
