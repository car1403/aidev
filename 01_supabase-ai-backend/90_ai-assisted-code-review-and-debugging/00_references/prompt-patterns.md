# Codex 요청 패턴

## 기본 패턴

```text
무엇을 만들지:
현재 상태:
원하는 결과:
제약 조건:
완료 기준:
```

## 코드 생성 요청

```text
Python으로 점수 리스트의 평균을 계산하는 함수를 만들어줘.

조건:
- 함수 이름은 calculate_average로 해줘.
- 점수 리스트를 인자로 받아.
- 평균을 반환해.
- 빈 리스트는 고려하지 않아도 돼.
```

## 코드 설명 요청

```text
아래 코드를 초급자 기준으로 설명해줘.
각 줄이 무슨 역할을 하는지 알려줘.
```

## 디버깅 요청

```text
아래 명령을 실행했더니 오류가 났어.

실행 명령:
python starter.py

오류 메시지:
...

내가 기대한 결과:
...

원인을 설명하고 수정 방향을 제안해줘.
```

## 리팩토링 요청

```text
아래 코드를 함수로 나누고 변수명을 더 명확하게 바꿔줘.
동작은 바꾸지 말아줘.
```

## 코드 리뷰 요청

```text
아래 코드를 리뷰해줘.
우선순위는 버그 가능성, 읽기 쉬움, 중복 제거, 예외 처리 순서로 봐줘.
```
