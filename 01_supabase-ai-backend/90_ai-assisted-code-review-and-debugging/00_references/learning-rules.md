# AI 보조 개발 학습 규칙

Codex는 개발을 대신 끝내는 도구가 아니라, 코드를 이해하고 더 안전하게 수정하도록 돕는 도구입니다.

## Codex가 도와줄 수 있는 일

- 코드 예시 만들기
- 오류 원인 설명하기
- 리팩토링 제안하기
- 테스트 케이스 제안하기
- README 초안 만들기
- 코드 리뷰와 보안 점검 기준 제안하기

## 반드시 직접 해야 하는 일

- 요구사항 직접 정리하기
- 생성된 코드 읽기
- 직접 실행하기
- 모르는 코드 질문하기
- 오류 메시지 확인하기
- API key와 token 노출 여부 확인하기
- 본인 말로 설명하기

## 피해야 할 행동

- 에러 메시지를 읽지 않고 바로 수정 요청하기
- 전체 과제를 한 번에 대신 작성하게 하기
- 이해하지 못한 코드를 그대로 사용하기
- API key나 비밀번호를 요청문에 넣기
- 실행 검증 없이 결과를 믿기
- Codex의 리뷰 결과를 요구사항 확인 없이 모두 적용하기
