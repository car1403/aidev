# AI 보조 개발 학습 규칙

## Codex가 해도 되는 일

- 코드 예시 만들기
- 오류 원인 설명하기
- 리팩토링 제안하기
- 테스트 케이스 제안하기
- README 초안 만들기

## 필수 실천 항목

- 요구사항 직접 정리하기
- 생성된 코드 읽기
- 직접 실행하기
- 모르는 코드 질문하기
- 본인 말로 설명하기

## 피해야 할 행동

- 에러 메시지를 읽지 않고 바로 수정 요청하기
- 전체 과제를 한 번에 대신 작성하게 하기
- 이해하지 못한 코드를 제출하기
- API Key나 비밀번호를 프롬프트에 넣기
- 실행 검증 없이 결과를 믿기
