# 02_ch2_prompting-for-code

코드 생성을 요청할 때 필요한 정보를 구조화하는 방법을 학습합니다.

## 학습 내용

- 기능 설명 작성
- 입력과 출력 정의
- 제약 조건 작성
- 완료 기준 작성
- 큰 요구사항을 작은 작업으로 나누기

## 좋은 요청 예시

```text
Python 함수 calculate_average를 만들어줘.

입력:
- 숫자 리스트 scores

출력:
- 평균 점수

조건:
- 빈 리스트는 처리하지 않아도 돼.
- sum과 len을 사용해줘.

완료 기준:
- calculate_average([90, 80, 70]) 결과가 80.0이어야 해.
```
