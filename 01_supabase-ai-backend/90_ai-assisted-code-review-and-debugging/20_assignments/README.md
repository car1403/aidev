# 20_assignments

Codex 활용 개발 과제입니다.

## 과제 목록

```text
assignment01_codex-learning-log
assignment02_prompt-improvement
assignment03_debugging-report
assignment04_backend-code-review-report
assignment05_security-cost-review-report
```

## 제출 기준

- 사용한 프롬프트를 기록한다.
- Codex 응답을 그대로 붙이지 않고 본인 말로 정리한다.
- 생성된 코드가 있다면 직접 실행 결과를 포함한다.
- 무엇을 배웠고 무엇을 검증했는지 적는다.

## 백엔드 리뷰 과제

### Assignment 04 - Backend Code Review Report

제출 내용:

- 리뷰 대상 파일
- 사용한 Codex 리뷰 프롬프트
- 발견한 문제
- 실제 수정한 내용
- 수정 후 실행 결과
- 아직 수정하지 않은 내용과 이유

### Assignment 05 - Security And Cost Review Report

제출 내용:

- 점검한 secret 목록
- `.env`와 `.env.example` 관리 상태
- 실제 LLM API 호출 가능성이 있는 파일
- 비용 제한 장치(`max_tokens`, mock 우선 실행 등)
- Supabase service role key 노출 여부
- Upstash Redis token 노출 여부
- 서비스 로그의 민감 정보 포함 가능성
