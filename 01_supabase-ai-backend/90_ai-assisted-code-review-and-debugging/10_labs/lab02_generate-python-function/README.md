# Lab 02 - Python 함수 생성 요청

이 lab에서는 Codex에게 명확한 요구사항을 전달하여 Python 함수를 생성하는 방법을 연습합니다.

좋은 코드 생성 결과를 얻으려면 "무엇을 만들지"뿐만 아니라 "입력, 출력, 예외 상황, 완료 기준"을 함께 알려주어야 합니다.

## 실습 목표

- 함수 생성 요청을 구체적으로 작성할 수 있다.
- 입력값과 반환값을 명확히 설명할 수 있다.
- 빈 리스트 같은 예외 상황을 요구사항에 포함할 수 있다.
- 생성된 코드가 요구사항을 만족하는지 테스트할 수 있다.

## 실습 파일

```text
starter.py
```

## 실습 요구사항

Codex에게 다음 함수를 만들도록 요청합니다.

- 함수명: `calculate_average`
- 입력: 숫자 리스트
- 출력: 평균값
- 조건: 빈 리스트가 들어오면 `0` 반환
- 조건: 원본 리스트를 변경하지 않음
- 테스트: 최소 3개 이상의 입력값으로 확인

## Codex 요청 예시

```text
Python 함수 하나를 작성해주세요.

요구사항:
1. 함수명은 calculate_average입니다.
2. 숫자 리스트를 입력받습니다.
3. 리스트 안 숫자의 평균을 반환합니다.
4. 빈 리스트가 들어오면 0을 반환합니다.
5. 원본 리스트는 변경하지 않습니다.
6. 초보자가 이해할 수 있도록 주석을 간단히 달아주세요.

추가 요청:
- 테스트 예시 3개를 함께 작성해주세요.
- 각 테스트가 어떤 상황을 확인하는지도 설명해주세요.
```

## 검토 기준

생성된 코드가 아래 기준을 만족하는지 확인합니다.

- 함수명이 요구사항과 일치하는가?
- 빈 리스트에서 오류가 나지 않는가?
- 평균 계산이 정확한가?
- 반환값이 숫자인가?
- 테스트 예시가 정상, 빈 리스트, 여러 숫자 상황을 포함하는가?
- 주석이 너무 많거나 너무 부족하지 않은가?

## 완료 체크리스트

- [ ] 입력과 출력을 명확히 작성했다.
- [ ] 빈 리스트 조건을 요구사항에 포함했다.
- [ ] 테스트 예시를 3개 이상 실행했다.
- [ ] 생성된 코드가 요구사항과 일치하는지 확인했다.
