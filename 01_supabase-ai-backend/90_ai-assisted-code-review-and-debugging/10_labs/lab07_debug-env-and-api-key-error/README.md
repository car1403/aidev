# Lab 07 - 환경변수와 API Key 오류 디버깅

## 목표

`.env` 설정 누락, 변수명 오타, key 출력 위험을 Codex와 함께 점검합니다.

## 실습 파일

```text
broken_env_loader.py
```

## 실습 절차

1. `broken_env_loader.py`를 읽습니다.
2. 어떤 변수명이 잘못되었는지 찾습니다.
3. key를 그대로 출력하는 코드가 왜 위험한지 설명합니다.
4. Codex에게 디버깅 요청 템플릿으로 질문합니다.
5. 수정안을 작성합니다.

## 요청 예시

```text
아래 환경변수 로딩 코드에 문제가 있습니다.
실제 key 값은 붙이지 않겠습니다.

문제:
Supabase URL이 없다고 나오고, API key가 그대로 출력될 위험이 있습니다.

원인과 수정 방법을 초보자 기준으로 설명해줘.
```
