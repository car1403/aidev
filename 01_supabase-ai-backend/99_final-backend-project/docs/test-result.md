## 테스트 항목

| 항목 | 확인 내용 | 결과 |
| --- | --- | --- |
| 서버 실행 | `GET /health`가 200을 반환하는가? |  |
| 질문 생성 | `POST /qa`가 201 또는 200을 반환하는가? |  |
| 질문 목록 | `GET /qa`가 목록을 반환하는가? |  |
| 단건 조회 | `GET /qa/{id}`가 1개 데이터를 반환하는가? |  |
| 로그 조회 | `GET /service-logs`가 로그 목록을 반환하는가? |  |
| Supabase 저장 | Table Editor에서 데이터가 보이는가? |  |
| 오류 처리 | 잘못된 요청에 적절한 오류가 나오는가? |  |
