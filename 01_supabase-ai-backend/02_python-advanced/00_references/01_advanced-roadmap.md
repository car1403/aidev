# 01. Advanced Roadmap

## 학습 흐름

```text
함수 심화
-> 모듈과 패키지
-> 예외 처리
-> 클래스와 객체
-> 컴프리헨션과 이터레이터
-> 데이터 처리
-> API
-> 비동기 처리와 외부 API 동시 호출
-> 테스트
-> 프로젝트 구조
```

## 목표

이 과정을 마치면 다음을 할 수 있어야 합니다.

```text
함수로 코드를 나누기
모듈을 만들어 import하기
예외가 발생해도 안전하게 처리하기
클래스로 데이터와 동작 묶기
CSV/JSON 데이터 다루기
API 응답을 처리하기
async/await로 여러 외부 API 요청을 동시에 처리하기
테스트 코드 작성하기
작은 프로젝트 구조 만들기
```
