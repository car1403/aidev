# Assignment 99 - DB/세션 미니 프로젝트

## 목표

FastAPI, PostgreSQL, Redis를 사용해 백엔드 서비스의 데이터/세션/캐시 기능을 구현합니다.

## 필수 요구사항

1. PostgreSQL 컨테이너 실행
2. Redis 컨테이너 실행
3. 사용자 테이블 설계
4. 회원가입/로그인 API
5. 보호된 로그 생성 API
6. 사용자별 대화 이력 저장
7. 사용자 프로필 캐싱
8. README에 실행 명령과 테스트 결과 기록

## 제외 범위

`docker compose`는 사용하지 않습니다. 프로젝트 통합 단계에서 다룹니다.
