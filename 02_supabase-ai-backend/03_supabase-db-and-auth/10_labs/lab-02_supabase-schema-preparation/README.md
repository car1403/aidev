# Lab 02 - Supabase 테이블 준비

이 실습은 Python 코드를 실행하기 전에 Supabase 테이블 구조를 준비하는 단계입니다.

## 학습 목표

- SQL Editor에서 SQL 파일을 실행할 수 있습니다.
- 테이블, 컬럼, 기본키, 생성일 컬럼의 역할을 설명할 수 있습니다.
- 코드 오류가 아니라 테이블 미생성 때문에 발생하는 오류를 구분할 수 있습니다.

## 진행 방법

1. Supabase Dashboard에 접속합니다.
2. 실습 프로젝트를 선택합니다.
3. 왼쪽 메뉴에서 `SQL Editor`를 엽니다.
4. 아래 파일 내용을 복사해서 실행합니다.

```text
C:\aidev\02_supabase-ai-backend\03_supabase-db-and-auth\00_references\supabase-schema.sql
```

## 확인 기준

- `learning_notes` 테이블이 생성되어 있습니다.
- 대화 이력 실습을 위한 `conversations`, `messages`, `service_logs` 테이블이 생성되어 있습니다.
- SQL Editor에서 실행 결과가 성공으로 표시됩니다.
