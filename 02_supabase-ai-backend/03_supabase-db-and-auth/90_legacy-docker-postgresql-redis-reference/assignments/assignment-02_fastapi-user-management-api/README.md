# Assignment 02 - FastAPI 사용자 관리 API

FastAPI와 로컬 PostgreSQL을 연결해 사용자 관리 API를 설계하는 legacy 과제 예시입니다.

## 요구사항

1. 사용자 생성 API를 설계합니다.
2. 사용자 목록 조회 API를 설계합니다.
3. 사용자 수정과 삭제 API를 설계합니다.
4. Swagger UI 테스트 결과를 기록합니다.

## 현재 과정에서의 대응

현재 과정에서는 FastAPI와 Supabase를 연결해 notes API를 구현합니다.
