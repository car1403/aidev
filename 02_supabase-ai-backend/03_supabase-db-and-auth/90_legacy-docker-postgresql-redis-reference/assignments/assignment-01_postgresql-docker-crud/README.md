# Assignment 01 - PostgreSQL Docker CRUD

Docker PostgreSQL에서 CRUD를 수행하는 legacy 과제 예시입니다.

현재 공식 과제는 Supabase 기준의 `20_assignments`를 사용합니다.

## 요구사항

1. PostgreSQL 컨테이너를 실행합니다.
2. `users`, `service_logs` 테이블을 만듭니다.
3. `INSERT`, `SELECT`, `UPDATE`, `DELETE`를 실행합니다.
4. 실행 명령과 결과를 README에 기록합니다.

## 현재 과정에서의 대응

현재 과정에서는 Supabase SQL Editor와 Supabase SDK로 같은 CRUD 개념을 학습합니다.
