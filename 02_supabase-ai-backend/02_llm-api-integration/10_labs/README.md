# 10_labs

LLM API 연동 단원의 실습 모음입니다.

## 권장 순서

1. `lab-01_llm-message-and-parameters`: 메시지와 파라미터 설계
2. `lab-02_api-key-safety-check`: API Key 보안 점검
3. `lab-03_single-turn-mock-to-gemini-sdk`: mock에서 Gemini SDK로 확장
4. `lab-04_multi-turn-mock-to-gemini-sdk`: 대화 이력 기반 호출
5. `lab-05_fastapi-llm-endpoint`: FastAPI 엔드포인트 연결
6. `lab-99_llm-integration-review`: 전체 점검

먼저 mock으로 구조를 확인한 뒤 실제 API 호출을 진행합니다.
