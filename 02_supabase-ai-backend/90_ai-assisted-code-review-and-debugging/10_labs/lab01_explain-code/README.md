# Lab 01 - 코드 설명 요청

이 lab에서는 Codex에게 Python 코드를 설명하게 하고, 설명이 실제 코드와 맞는지 확인합니다.

초보 단계에서는 코드를 무작정 수정하기 전에 먼저 "이 코드가 어떤 순서로 실행되는지"를 이해하는 것이 중요합니다.

## 실습 목표

- Codex에게 코드 설명을 요청할 수 있다.
- 변수, 조건문, 반복문, 함수가 어떤 역할을 하는지 설명을 들을 수 있다.
- Codex의 설명을 그대로 받아들이지 않고 코드와 비교할 수 있다.
- 이해한 내용을 본인 말로 다시 정리할 수 있다.

## 실습 파일

```text
sample.py
```

## 실습 절차

1. `sample.py` 파일을 엽니다.
2. 코드를 먼저 직접 읽어 봅니다.
3. 이해되는 부분과 모르는 부분을 나눕니다.
4. Codex에게 코드 설명을 요청합니다.
5. Codex가 설명한 내용이 실제 코드와 맞는지 줄별로 비교합니다.
6. 모르는 용어가 나오면 다시 질문합니다.
7. 마지막에 코드의 전체 흐름을 3~5문장으로 정리합니다.

## Codex 요청 예시

```text
아래 Python 코드를 초보자 기준으로 설명해주세요.

요청 사항:
1. 전체 코드가 무엇을 하는지 먼저 설명해주세요.
2. 각 줄이 어떤 역할을 하는지 설명해주세요.
3. 변수에 어떤 값이 들어가는지 예시를 들어주세요.
4. 조건문 또는 반복문이 있다면 실행 흐름을 순서대로 설명해주세요.
5. 마지막에 이 코드에서 꼭 기억해야 할 개념을 정리해주세요.

아직 코드를 수정하지 말고 설명만 해주세요.
```

## 완료 체크리스트

- [ ] 설명과 실제 코드를 비교했다.
- [ ] 모르는 용어를 다시 질문했다.
- [ ] 코드 흐름을 본인 말로 정리했다.
