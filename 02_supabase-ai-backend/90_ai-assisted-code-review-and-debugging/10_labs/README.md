# 10_labs

이 폴더는 앞 단원에서 자주 나오는 오류를 짧게 분석해 보는 실습입니다.

| Lab | 주제 |
| --- | --- |
| `lab-01_traceback-and-import-error.md` | import 오류와 가상환경 확인 |
| `lab-02_env-and-api-key-error.md` | `.env`, API key, 모델 오류 확인 |
| `lab-03_fastapi-404-405-validation.md` | 404, 405, 422 오류 구분 |
| `lab-04_supabase-redis-errors.md` | Supabase, Auth/JWT, Redis 오류 확인 |

실습 목적은 오류를 모두 외우는 것이 아니라, 오류를 정리해서 질문하는 습관을 만드는 것입니다.
