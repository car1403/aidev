# 03. Prompt Recipes

Codex에게 요청할 때는 “무엇을”, “어디까지”, “어떤 기준으로” 봐야 하는지 함께 적습니다.

## 오류 분석 요청

```text
아래 FastAPI 오류를 초보자 기준으로 분석해주세요.

파일:
C:\aidev\02_supabase-ai-backend\...

실행 명령:
...

오류 메시지:
...

요청:
1. 가능한 원인을 우선순위로 정리해주세요.
2. 확인 명령을 알려주세요.
3. 아직 파일은 수정하지 말아주세요.
```

## 코드 리뷰 요청

```text
아래 파일을 리뷰해주세요.

리뷰 관점:
1. 실행 오류 가능성
2. FastAPI endpoint 설계
3. Pydantic 검증
4. Supabase 저장 흐름
5. API key와 로그 보안

요청:
치명도 순서로 문제를 먼저 알려주고, 수정 제안은 그 다음에 알려주세요.
```

## 최종 프로젝트 사전 점검 요청

```text
99_final-backend-project 제출 전 점검을 해주세요.

확인할 것:
1. 필수 endpoint가 있는지
2. Supabase 저장 흐름이 있는지
3. mock 또는 Gemini 호출이 구분되어 있는지
4. .env가 노출되지 않았는지
5. Swagger 테스트 결과를 제출 문서에 남겼는지

아직 코드를 수정하지 말고 누락된 항목만 알려주세요.
```
