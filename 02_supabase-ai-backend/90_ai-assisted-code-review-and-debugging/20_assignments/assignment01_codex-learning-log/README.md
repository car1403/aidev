# Assignment 01 - Codex 학습 기록

이 과제에서는 Codex를 사용한 과정을 기록하고, 어떤 질문이 도움이 되었는지 스스로 정리합니다.

Codex를 잘 사용하는 능력은 한 번에 생기지 않습니다. 어떤 요청을 했을 때 답변이 좋았는지, 어떤 요청은 모호했는지 기록해 두면 다음 질문을 더 정확하게 만들 수 있습니다.

## 목표

- Codex에게 사용한 요청문을 기록할 수 있다.
- 요청의 목적과 결과를 구분할 수 있다.
- Codex 답변 중 직접 확인한 내용을 정리할 수 있다.
- 다음 요청을 더 좋게 만드는 방법을 생각할 수 있다.

## 수행 절차

1. 오늘 사용한 Codex 요청문을 3개 이상 고릅니다.
2. 각 요청문이 어떤 목적이었는지 적습니다.
3. Codex 답변 중 실제로 도움이 된 부분을 정리합니다.
4. Codex 답변을 직접 확인한 방법을 적습니다.
5. 같은 요청을 다시 한다면 어떻게 바꿀지 작성합니다.
