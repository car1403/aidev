# Docker Is Covered Later

이 통합 실습은 Supabase 기반으로 진행합니다.

03 과정에서는 Docker Compose를 사용하지 않습니다. 데이터 저장은 Supabase 클라우드 프로젝트를 사용하고, 로컬 PC에서는 FastAPI와 Streamlit만 실행합니다.

```text
Streamlit
-> FastAPI
-> Supabase
```

Docker, Docker Compose, AWS, GitHub Actions, 운영 모니터링은 `06_multi-agent-service-ops`에서 학습합니다.
