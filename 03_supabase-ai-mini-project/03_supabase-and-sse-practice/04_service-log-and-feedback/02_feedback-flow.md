# 02. 사용자 피드백 흐름

사용자 피드백은 AI 응답 품질을 개선하기 위한 데이터입니다.

## 기본 흐름

```text
AI 응답 표시
-> 사용자가 평점과 의견 입력
-> Streamlit이 FastAPI API 호출
-> FastAPI가 feedbacks 테이블에 저장
-> 대시보드에서 피드백 목록 조회
```

## 피드백 컬럼 예시

```text
message_id
-> 어떤 AI 응답에 대한 평가인지 연결

rating
-> 1~5 점수

comment
-> 자유 의견

created_at
-> 피드백 작성 시간
```

## 활용 방법

- 낮은 점수를 받은 응답을 다시 확인합니다.
- 자주 나오는 불만을 기능 개선 항목으로 정리합니다.
- 발표 자료에서 개선 전후를 설명할 수 있습니다.

## 체크리스트

- [ ] 어떤 응답에 대한 피드백인지 연결할 수 있다.
- [ ] 평점 범위를 정했다.
- [ ] 의견 입력은 선택 또는 필수 중 하나로 정했다.
- [ ] 피드백 저장 성공/실패 메시지를 화면에 표시한다.
