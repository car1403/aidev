# Setup Supabase

Supabase 학습 로그 샘플을 실행하기 위한 테이블 준비 문서입니다.

## 1. Create Project

Supabase 프로젝트를 생성합니다.

## 2. Create Table

Supabase SQL Editor에서 다음 SQL을 실행합니다.

```sql
create table if not exists public.learning_logs (
 id bigint generated by default as identity primary key,
 learner_name text not null,
 topic text not null,
 minutes integer not null check (minutes > 0),
 status text not null default 'created',
 memo text default '',
 created_at timestamptz not null default now()
);
```

## 3. Seed Data

샘플 데이터를 넣습니다.

```sql
insert into public.learning_logs (learner_name, topic, minutes, status, memo)
values
 ('Kim', 'Supabase Mini Project', 60, 'done', 'Supabase table is ready.'),
 ('Lee', 'FastAPI Integration', 45, 'doing', 'Health check API is ready.'),
 ('Park', 'Streamlit Dashboard', 50, 'created', 'Dashboard screen will be connected.');
```

## 4. Create.env

`.env.example`을 복사해 `.env`를 만들고 실제 값을 입력합니다.

```text
SUPABASE_URL=
SUPABASE_ANON_KEY=
API_BASE_URL=http://127.0.0.1:8000
```

service role key가 필요한 팀 프로젝트는 백엔드에서만 사용합니다.
