# 01. Topic Selection

07의 기본 주제는 **에러 자가 치유(Auto Healing) 워크플로우**입니다.

팀 프로젝트에서는 이 큰 주제 안에서 어떤 장애를 다룰지 정합니다.

## 주제 예시

| 주제 | 설명 |
| --- | --- |
| API 장애 자동 복구 | backend 5xx, timeout, 응답 지연을 감지하고 retry/fallback 처리 |
| LLM 응답 품질 복구 | 빈 응답, 형식 오류, 정책 위반 응답을 감지하고 재요청 |
| Worker 장애 복구 | worker 작업 실패를 감지하고 재시도 또는 상태 재처리 |
| 운영 로그 기반 장애 분석 | 로그를 기반으로 장애 원인을 분류하고 대시보드에 표시 |
| Prompt Injection 방어 | 위험 지시문을 감지하고 요청을 차단하거나 별도 로그 기록 |

## 주제 선택 기준

- Docker Compose로 실행 가능한가?
- 장애 이벤트를 입력하거나 시뮬레이션할 수 있는가?
- Agent 역할을 나눌 수 있는가?
- 복구 전략을 2개 이상 설명할 수 있는가?
- Health Check 또는 결과 검증이 가능한가?
- monitor 화면에 결과를 표시할 수 있는가?
