# FastAPI는 backend API 서버를 만들 때 사용합니다.
fastapi

# uvicorn은 FastAPI 앱을 컨테이너 안에서 실행하는 ASGI 서버입니다.
uvicorn

# Streamlit은 frontend 화면과 monitor 대시보드를 만들 때 사용합니다.
streamlit

# httpx는 서비스 간 HTTP 요청을 보낼 때 사용합니다.
httpx

# pydantic은 장애 이벤트, Agent Context, 복구 결과 데이터를 검증할 때 사용합니다.
pydantic

# python-dotenv는 .env 파일의 환경변수를 Python 코드에서 읽을 때 사용합니다.
python-dotenv
