# 10 Labs

이 폴더는 Docker, Docker Compose, GitHub Actions, AWS 배포 흐름을 단계별로 실습하는 lab 문서입니다.

## Lab 목록

| Lab | 내용 |
| --- | --- |
| `lab-01_docker-build-and-run.md` | 단일 FastAPI 서비스 Docker build/run |
| `lab-02_docker-compose-up.md` | Docker Compose 멀티 서비스 실행 |
| `lab-03_health-check-and-logs.md` | Health Check와 로그 확인 |
| `lab-04_github-actions-docker-build.md` | GitHub Actions Docker build 검증 |
| `lab-05_aws-deployment-checklist.md` | AWS 배포 전 체크리스트 |

## 공통 제출 기준

- 실행한 명령어
- 실행 결과 화면 또는 출력
- 오류가 있었다면 해결 과정
- 운영 관점에서 확인한 내용
