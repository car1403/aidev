# Assignment 01. AI 워크플로우 시나리오 설계

## 목표

하나의 업무 자동화 주제를 선택하고, AI 워크플로우 구조로 설계합니다.

## 선택 주제 예시

아래 중 하나를 선택하거나 직접 정해도 됩니다.

```text
1. 고객 문의 자동 분류 및 답변 초안 생성
2. 회의록 요약 및 후속 작업 생성
3. 채용 지원서 검토 및 후보자 요약
4. 장애 리포트 분석 및 운영팀 알림
5. 기술 지원 문서 검색 기반 답변 생성
```

## 작성 항목

다음 항목을 반드시 포함합니다.

```text
1. 업무 배경
2. 자동화가 필요한 이유
3. Trigger
4. Input 데이터
5. Condition 분기
6. LLM 역할
7. Tool 역할
8. Memory에 저장할 정보
9. Action 목록
10. Output
11. Log와 운영 확인 항목
12. 오류 발생 시 대응 방법
```

## 제출 형식

Markdown 문서로 작성합니다.

```text
제목:
업무 배경:
워크플로우 단계:
LLM/Tool/Memory 역할:
예상 오류:
운영 확인 방법:
```

## 평가 기준

- Trigger, Condition, Action이 명확한가?
- LLM이 필요한 단계와 규칙 기반 처리가 가능한 단계를 구분했는가?
- Tool과 Memory 역할을 과도하게 LLM에 맡기지 않았는가?
- 결과 검증과 로그 기록을 고려했는가?
- 이후 AIPP, n8n, Dify 중 하나로 구현할 수 있을 만큼 단계가 구체적인가?
