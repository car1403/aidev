# Assignment 01. n8n 기반 AI 워크플로우 설계

## 목표

n8n을 사용해 구현할 수 있는 AI 업무 자동화 워크플로우를 설계합니다.

## 주제 예시

```text
1. 고객 문의 접수 후 AI 답변 초안 생성
2. 장애 리포트 접수 후 운영팀 알림
3. 회의록 업로드 후 요약과 액션 아이템 생성
4. 신규 리드 접수 후 영업 우선순위 분류
5. 기술 지원 문서 검색 후 응답 생성
```

## 필수 포함 항목

```text
1. 업무 배경
2. Trigger 노드
3. 입력 JSON 예시
4. Set 노드에서 정리할 필드
5. IF 또는 Switch 분기 조건
6. HTTP Request로 호출할 API
7. AI API 요청/응답 구조
8. 성공 시 Action
9. 실패 시 Action
10. 실행 로그와 운영 확인 항목
```

## 제출 형식

Markdown 문서로 작성합니다.

```text
# 프로젝트명

## 업무 배경
## n8n 노드 흐름
## 입력/출력 데이터
## AI API 연동
## 예외 처리
## 운영 확인 항목
```

## 평가 기준

- n8n 노드 흐름이 구체적인가?
- Trigger, Condition, Action 구분이 명확한가?
- AI API 입력과 출력 구조가 설계되어 있는가?
- 실패 흐름과 로그를 고려했는가?
- 이후 Dify 또는 99 미니 프로젝트로 확장 가능한가?
