# Assignment 01. Dify 기반 AI 워크플로우 설계

## 목표

Dify에서 구현할 수 있는 AI 앱 또는 워크플로우를 설계합니다.

## 주제 예시

```text
1. 사내 문서 검색 챗봇
2. 기술 지원 문의 답변 초안 생성기
3. 회의록 요약 및 액션 아이템 생성기
4. 고객 문의 분류 Workflow
5. n8n에서 호출할 Dify API 앱
```

## 필수 포함 항목

```text
1. 업무 배경
2. 앱 유형 선택: Chatflow 또는 Workflow
3. 선택 이유
4. 사용자 입력
5. System Prompt 목적
6. Knowledge 구성
7. Tool 또는 API 연동 필요 여부
8. 노드 흐름
9. 출력 형식
10. 실패 또는 검토 필요 조건
11. 운영 로그와 품질 점검 항목
```

## 제출 형식

Markdown 문서로 작성합니다.

```text
# 프로젝트명

## 업무 배경
## 앱 유형과 선택 이유
## 입력/출력
## Knowledge/RAG 설계
## Workflow 또는 Chatflow 흐름
## API 연동 방식
## 운영 확인 항목
```

## 평가 기준

- Chatflow와 Workflow 선택 기준이 명확한가?
- Knowledge/RAG 설계가 업무와 연결되어 있는가?
- 입력과 출력이 구체적인가?
- n8n 또는 외부 백엔드에서 호출 가능한 구조로 설계했는가?
- 실패 대응과 운영 로그를 고려했는가?
