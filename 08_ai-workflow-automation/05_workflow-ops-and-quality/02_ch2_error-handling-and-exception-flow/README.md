## 핵심 개념

```text
Retry: 같은 작업을 다시 시도
Fallback: 대체 경로 실행
Escalation: 사람 담당자에게 넘김
Timeout: 오래 걸리는 작업 중단
Validation Error: 입력이나 출력 형식 오류
```
