# 05_ch5_workflow-ops-assistant

운영 데이터를 바탕으로 워크플로우 개선안을 제안하는 Ops Assistant를 설계합니다.

## 핵심 개념

Workflow Ops Assistant는 실행 로그, 실패 유형, 비용, 사용자 피드백을 분석해 운영자가 다음 개선 작업을 정할 수 있게 도와주는 AI 보조 도구입니다.

```text
Logs
-> Metrics
-> Issue Detection
-> Recommendation
-> Next Action
```
