# 04_ch4_data-quality-validation

AI 워크플로우의 입력과 출력 데이터 품질을 검증하는 방법을 학습합니다.

## 핵심 개념

```text
Input Validation: 입력 데이터 검증
Output Validation: 출력 형식 검증
Policy Check: 금지 표현 또는 위험 내용 검토
Required Fields: 필수 필드 확인
Structured Output: JSON 등 정해진 형식의 출력
```
