# 03_ch3_log-analysis-and-improvement

워크플로우 실행 로그를 분석하고 개선 포인트를 찾는 방법을 학습합니다.

## 핵심 개념

```text
Execution Log: 실행 이력
Error Rate: 실패 비율
Latency: 응답 시간
User Feedback: 사용자 만족/불만
Improvement Loop: 분석 결과를 다음 버전에 반영하는 흐름
```
