# Assignment 01. AI 워크플로우 운영/품질 관리 계획

## 목표

01~04에서 설계한 AI 워크플로우 중 하나를 선택하고, 운영과 품질 관리 계획을 작성합니다.

## 필수 포함 항목

```text
1. 대상 워크플로우
2. 버전 관리 대상: model, prompt, workflow
3. 비용 추적 항목
4. 오류 유형 목록
5. Retry/Fallback/Escalation 흐름
6. 실행 로그 항목
7. 입력 데이터 검증 기준
8. 출력 데이터 검증 기준
9. 사용자 피드백 수집 방식
10. 개선 주기
11. Ops Assistant가 제안해야 할 항목
```

## 제출 형식

```text
# 워크플로우 운영/품질 관리 계획

## 워크플로우 개요
## 버전 관리
## 비용 관리
## 오류 처리
## 로그 분석
## 데이터 품질 검증
## 개선 루프
## Ops Assistant 설계
```

## 평가 기준

- 운영 중 발생할 수 있는 문제를 현실적으로 고려했는가?
- 비용, 오류, 품질, 로그 기준이 구체적인가?
- 자동 처리와 사람 검토 기준이 구분되어 있는가?
- 99 미니 프로젝트에 반영 가능한 수준으로 설계했는가?
