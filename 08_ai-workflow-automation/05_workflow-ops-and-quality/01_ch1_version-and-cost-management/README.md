# 01_ch1_version-and-cost-management

모델, 프롬프트, 워크플로우 버전과 비용을 관리하는 방법을 학습합니다.

## 핵심 개념

```text
Model Version: 어떤 모델을 사용했는가?
Prompt Version: 어떤 프롬프트를 사용했는가?
Workflow Version: 어떤 노드 구성을 사용했는가?
Cost: 실행당 비용 또는 토큰 사용량은 얼마인가?
Latency: 응답 시간은 얼마나 걸렸는가?
```
