# Assignment 01. AIPP식 AI 워크플로우 설계

## 목표

하나의 업무 자동화 주제를 선택하고, AIPP에서 구현할 수 있는 노드 기반 워크플로우로 설계합니다.

## 주제 예시

```text
1. 고객 문의 자동 분류 및 답변 초안 생성
2. 회의록 요약 후 업무 담당자 배정
3. 장애 리포트 접수 후 운영팀 알림
4. 기술 지원 문서 검색 기반 응답 생성
5. 영업 리드 분류 및 후속 메일 초안 작성
```

## 작성 항목

```text
1. 업무 배경
2. 자동화 목표
3. Trigger
4. 필요한 입력 데이터
5. 노드 목록
6. 각 노드의 입력과 출력
7. LLM 노드의 프롬프트 목적
8. Tool 노드가 호출할 기능
9. Condition 분기 기준
10. Action 결과
11. 실패 시 대체 흐름
12. 로그와 검토 항목
```

## 제출 형식

Markdown 문서로 작성합니다.

```text
# 프로젝트명

## 업무 배경
## 워크플로우 목표
## 노드 설계
## 데이터 흐름
## 예외 처리
## 운영 검토 항목
```

## 평가 기준

- 노드 역할이 명확한가?
- LLM, Tool, Condition, Action이 적절히 분리되어 있는가?
- 노드 간 데이터 흐름이 구체적인가?
- 오류나 검토가 필요한 상황을 고려했는가?
- 이후 n8n 또는 Dify로도 옮겨 구현할 수 있는 구조인가?
