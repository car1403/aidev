# AI Assisted Review Guidelines

## 좋은 요청에 포함할 정보

- 내가 실행한 명령어
- 기대한 결과
- 실제 결과
- 오류 메시지 전체
- 관련 코드 일부
- 이미 시도한 해결 방법

## 좋은 요청 예시

```text
아래 Streamlit 코드에서 session_state가 기대한 대로 유지되지 않습니다.
실행 명령은 streamlit run app.py 입니다.
버튼을 누르면 count가 증가해야 하는데 화면 갱신 후 값이 다시 0처럼 보입니다.
원인을 먼저 설명하고, 수정 방향을 단계별로 제안해 주세요.
```

## 피해야 할 요청

```text
이거 고쳐줘.
```

```text
앱을 전부 만들어줘.
```

```text
왜 안 돼?
```

## 검토 기준

AI 답변을 받을 때는 다음을 확인합니다.

- 내 코드의 실제 상황을 반영했는가?
- 실행 가능한 수정 방법인가?
- 새로운 라이브러리를 불필요하게 추가하지 않았는가?
- 보안상 위험한 방식으로 토큰이나 비밀번호를 처리하지 않는가?
- 내가 설명할 수 있는 수준의 수정인가?
