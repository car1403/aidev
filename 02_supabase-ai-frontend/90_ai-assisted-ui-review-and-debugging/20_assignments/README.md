- 최종 앱이 정상 실행되어야 합니다.
- API 호출, `session_state`, Authorization header, 캐시, 보안 키 노출 여부 중 최소 2개 항목을 점검합니다.
- `04_ai-chatbot-interface` 또는 `05_state-session-and-data`에서 만든 파일을 리뷰 대상으로 선택하는 것을 권장합니다.
