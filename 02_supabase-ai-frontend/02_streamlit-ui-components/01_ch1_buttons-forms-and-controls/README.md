# 01_ch1_buttons-forms-and-controls

버튼, 체크박스, 슬라이더, 폼처럼 사용자의 행동을 받는 UI 컴포넌트를 학습합니다.

## 예제 파일

```text
01_button-click.py
02_checkbox-slider.py
03_form-input.py
04_simple-survey-form.py
```

## 실행 예시

```powershell
streamlit run .\02_streamlit-ui-components\01_ch1_buttons-forms-and-controls\01_button-click.py
```

## 확인할 내용

- 버튼 클릭 후 결과가 표시되는가?
- 체크박스 값이 `True` 또는 `False`로 처리되는가?
- 슬라이더 값으로 조건문을 만들 수 있는가?
- 폼 제출 전과 제출 후의 화면 흐름을 구분할 수 있는가?
