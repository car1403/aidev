# Assignment 02: Table Chart Dashboard

## 목표

pandas DataFrame, 표, 차트를 사용해 기본 대시보드를 만듭니다.

## 요구사항

- 직접 만든 리스트 또는 딕셔너리 데이터를 DataFrame으로 변환합니다.
- `st.dataframe`으로 표를 출력합니다.
- 필터 입력을 하나 이상 제공합니다.
- `st.bar_chart` 또는 `st.line_chart`를 사용합니다.
- `st.metric`으로 요약 지표를 최소 1개 표시합니다.

## 제출 파일 예시

```text
assignment-02-table-chart-dashboard.py
```
