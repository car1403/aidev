# Assignment 02: Chat History App

## 목표

`st.session_state`를 사용해 대화 이력이 유지되는 챗봇 앱을 만듭니다.

## 요구사항

- 메시지 목록을 `st.session_state.messages`에 저장합니다.
- 각 메시지는 `role`, `content`를 포함합니다.
- 새 질문을 입력하면 사용자 메시지와 assistant 응답이 목록에 추가됩니다.
- 대화 초기화 버튼을 제공합니다.

## 제출 파일 예시

```text
assignment-02-chat-history-app.py
```
