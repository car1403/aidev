# Assignment 03: AI Service Client

## 목표

Streamlit 챗봇 화면에서 백엔드 mock chat service API를 호출하는 앱을 만듭니다.

## 요구사항

- API 기본 주소를 변수로 분리합니다.
- 질문을 `/api/chat`에 POST 요청으로 보냅니다.
- 응답 JSON의 `reply` 값을 assistant 메시지로 표시합니다.
- 연결 실패, 타임아웃, HTTP 오류를 처리합니다.
- 실제 AI 모델 대신 샘플 백엔드 API를 사용해도 됩니다.

## 제출 파일 예시

```text
assignment-03-ai-service-client.py
```
