# 10_labs

수업 중 함께 진행하는 실습 파일입니다.

## 실습 목록

```text
lab-01-profile-card-app.py
lab-02-learning-check-app.py
lab-03-simple-dashboard.py
```

## 실습 진행 방식

1. 예제 파일을 먼저 실행합니다.
2. 입력값을 바꾸며 화면 변화를 확인합니다.
3. 주석을 읽고 코드 흐름을 설명합니다.
4. 요구사항에 맞게 일부 코드를 직접 수정합니다.

## 실행 예시

```powershell
streamlit run .\01_streamlit-basic\10_labs\lab-01-profile-card-app.py
```
