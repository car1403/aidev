## 기본 템플릿

```text
llm-agent-team-template
```

팀별로 복사해서 사용합니다.

```text
team-01-learning-coach-agent
team-02-schedule-agent
team-03-document-qa-agent
```

## 진행 기준

- 템플릿을 그대로 실행해본 뒤 수정합니다.
- README와 테스트 체크리스트를 항상 같이 업데이트합니다.
- API Key는 `.env`에만 작성합니다.
