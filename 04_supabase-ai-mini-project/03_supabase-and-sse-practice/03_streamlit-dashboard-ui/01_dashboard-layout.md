# 01. 대시보드 화면 구성

대시보드는 데이터를 한눈에 확인하기 위한 화면입니다.

04 미니 프로젝트에서는 화려한 화면보다 다음 정보를 명확하게 보여주는 것이 중요합니다.

```text
현재 서비스 상태
대화 목록
최근 메시지
최근 서비스 로그
사용자 피드백
```

## 기본 화면 구성

```text
st.title()
-> 화면 제목

st.sidebar
-> 필터, 사용자 선택, 새로고침 버튼

st.form()
-> 질문 입력 또는 피드백 입력

st.dataframe()
-> 로그와 메시지 목록 표시

st.info(), st.success(), st.error()
-> 현재 상태와 오류 안내
```

## 설계 팁

- 화면 하나에 모든 기능을 넣기보다 영역을 나눕니다.
- 버튼을 누른 뒤 어떤 일이 일어났는지 메시지로 알려줍니다.
- 오류가 발생하면 개발자용 오류와 사용자용 메시지를 구분합니다.
- 표에는 너무 많은 컬럼을 한 번에 표시하지 않습니다.
