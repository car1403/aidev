# Docker Compose vs Supabase

04 과정은 Supabase managed PostgreSQL을 기본 데이터베이스로 사용합니다. Docker Compose로 PostgreSQL이나 Redis를 직접 운영하지 않습니다.

| 구분 | 04 과정 기준 | 이후 과정 |
| --- | --- | --- |
| Database | Supabase | Docker/PostgreSQL 운영 구조는 07에서 확장 |
| Redis | 필요 시 Upstash Redis 선택 | Docker Compose 서비스 운영은 07에서 학습 |
| 배포 | 선택 확장 | 07에서 GitHub Actions/AWS 흐름 학습 |
