# Test Checklist Template

프로젝트 테스트 체크리스트입니다.

## 1. 로컬 실행

- [ ] `.venv`가 활성화된다.
- [ ] FastAPI가 실행된다.
- [ ] Streamlit이 실행된다.
- [ ] `.env` 값을 정상적으로 읽는다.

## 2. API 테스트

- [ ] `/health`가 정상 응답한다.
- [ ] 로그 생성 API가 동작한다.
- [ ] 로그 목록 조회 API가 동작한다.
- [ ] 로그 상세 조회 API가 동작한다.
- [ ] 피드백 저장 API가 동작한다.
- [ ] 잘못된 요청에 대해 오류 응답이 나온다.

## 3. Supabase 테스트

- [ ] 테이블이 생성되어 있다.
- [ ] 테스트 데이터가 들어간다.
- [ ] FastAPI에서 데이터를 조회한다.
- [ ] FastAPI에서 데이터를 저장한다.
- [ ] RLS를 사용하는 경우 정책 테스트를 했다.

## 4. Streamlit 테스트

- [ ] 화면이 열린다.
- [ ] 버튼 클릭 후 API가 호출된다.
- [ ] 데이터가 표 또는 차트로 표시된다.
- [ ] 오류 상황에서 안내 메시지가 표시된다.
- [ ] 피드백 입력이 저장된다.

## 5. 선택 확장 테스트

- [ ] SSE 응답이 화면에 누적 표시된다.
- [ ] 최종 assistant 응답만 Supabase에 저장된다.
- [ ] 배포 URL에서 화면이 열린다.
- [ ] 배포 환경 변수 설정을 확인했다.
