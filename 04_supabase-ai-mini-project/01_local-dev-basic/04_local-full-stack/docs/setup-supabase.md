# Setup Supabase

Supabase 학습 로그 샘플을 실행하기 위한 테이블 준비 문서입니다.

이 문서는 `04_local-full-stack` 예제를 실행하기 전에 Supabase에서 무엇을 준비해야 하는지 안내합니다.

## 1. Supabase 프로젝트 준비

Supabase Dashboard에서 프로젝트를 선택하거나 새로 만듭니다.

필요한 값은 아래 메뉴에서 확인합니다.

```text
Project Settings
-> API
```

확인할 값:

```text
SUPABASE_URL
SUPABASE_ANON_KEY
SUPABASE_SERVICE_ROLE_KEY
```

`SUPABASE_SERVICE_ROLE_KEY`는 백엔드에서만 사용합니다. Streamlit 프론트엔드 코드나 화면에는 넣지 않습니다.

## 2. 테이블 만들기

Supabase SQL Editor에서 다음 SQL을 실행합니다.

```sql
create table if not exists public.learning_logs (
  id bigint generated by default as identity primary key,
  learner_name text not null,
  topic text not null,
  minutes integer not null check (minutes > 0),
  status text not null default 'created',
  memo text default '',
  created_at timestamptz not null default now()
);
```

## 3. 샘플 데이터 넣기

샘플 데이터를 넣으면 Streamlit 화면에서 바로 조회 결과를 확인할 수 있습니다.

```sql
insert into public.learning_logs (learner_name, topic, minutes, status, memo)
values
  ('Kim', 'Supabase Mini Project', 60, 'done', 'Supabase table is ready.'),
  ('Lee', 'FastAPI Integration', 45, 'doing', 'Health check API is ready.'),
  ('Park', 'Streamlit Dashboard', 50, 'created', 'Dashboard screen will be connected.');
```

이미 샘플 데이터를 넣었다면 중복으로 넣지 않아도 됩니다.

## 4. .env 파일 준비

`04_supabase-ai-mini-project` 최상위에서 `.env.example`을 `.env`로 복사합니다.

```powershell
cd C:\aidev\04_supabase-ai-mini-project
Copy-Item .env.example .env
```

사용하는 `.env` 위치:

```text
C:\aidev\04_supabase-ai-mini-project\.env
```

필수 값:

```env
SUPABASE_URL=https://your-project-ref.supabase.co
SUPABASE_ANON_KEY=your-supabase-anon-key
SUPABASE_SERVICE_ROLE_KEY=your-supabase-service-role-key
API_BASE_URL=http://127.0.0.1:8000
```

Gemini와 OpenAI 값은 AI 응답 생성 실습에서 사용합니다. 이 로컬 풀스택 기본 예제는 먼저 Supabase 저장과 조회가 되는지 확인하는 단계입니다.

## 5. RLS 참고

이 예제는 초보자 로컬 실습용이므로 FastAPI 백엔드가 Supabase와 통신합니다.

권한 정책을 단순하게 확인하려면 다음 두 가지 중 하나를 선택합니다.

```text
1. 백엔드에서 SUPABASE_SERVICE_ROLE_KEY 사용
2. anon key를 사용할 경우 learning_logs 테이블에 적절한 RLS 정책 작성
```

실제 팀 프로젝트에서는 어떤 권한으로 어떤 작업을 허용할지 데이터베이스 설계서와 함께 정리합니다.
